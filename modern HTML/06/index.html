Comparison of HTML vs HTML5

HTML HTML5
Uses SGML (Standard Generalized Markup Language) for document structure. Uses XML (Extensible Markup Language) for document structure, which is stricter and more organized.
Limited multimedia support; relies on third-party plugins like Flash for audio and video. Native support for multimedia elements through <audio> and <video> tags.
Does not support local storage natively. Introduces local storage options like Web Storage (localStorage and sessionStorage) for storing data on the client side.
